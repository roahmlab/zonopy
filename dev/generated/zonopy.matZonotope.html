Bases: object

2D Matrix Zonotope class for representing a zonotope in matrix form.

The matrix zonotope is defined as a set of linear combinations of a center vector and generator matrix. Similar to the zonotope class, the matrix zonotope is defined as a set of linear combinations of a center vector and generator matrix. In this case, it is a set of the form:

Z={C+∑i=1NaiGi∣ai∈[0,1]}\mathcal{Z} = \{ C + \sum_{i=1}^{N} a_i G_i \mid a_i \in [0,1] \} Z={C+i=1∑N​ai​Gi​∣ai​∈[0,1]}

where CCC is the center matrix and GiG_iGi​ are the generator matrices.

Here, we define Z\mathbf{Z}Z as a tensor of shape (N+1)×dx×dy(N+1) \times dx \times dy(N+1)×dx×dy where dxdxdx and dydydy are the number of rows and columns of all matrices, respectively. That is, Z=[C,G1,G2,…,GN]\mathbf{Z} = [C, G_1, G_2, \ldots, G_N]Z=[C,G1​,G2​,…,GN​].
